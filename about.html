About Me

My name is Ivan Rodriguez and I am a software developer specializing in web and mobile. I attended the University of California, Berkeley where I studied economics and computer science.

I am currently working on personal projects with Ruby on Rails, PHP, and Objective-C.

While at Berkeley I spent time working at various on-campus locations programming internal tools for university staff. I also spent some time at Zynga working on the Zynga.org team.

Development

Sublime Text 2

Github

I use git and Github every day. It's a valuable tool to keep track of what I'm working on and store and distribute everything I've built.

I'm a recent convert to CodeKit, it's a great tool that auto-updates the CSS you're working with and reloads the browser.

Xcode

Xcode is by far the most beautiful IDE I've ever used. It's developed by Apple and it makes creating iOS apps a joy.

Eclipse

Eclipse is a bit of a beast but it's definitely my go to tool whenever I'm working with Java and its error checking tools are next to none.
